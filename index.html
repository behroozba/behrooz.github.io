Diamonds: Real-World Machine Learning Model

First, we pick Diamonds dataset from Kaggle. This dataset contains the information such as carat, cut, color, clarity, depth, table, price, x, y, and z for more than 54000 diamonds. After reading the dataset description, understand the problem statement, we perform exploratory data analysis, gather insights about the data, perform feature engineering, create a training-validation split, and prepare the data for modeling. Then we train & evaluate different machine learning models, tune hyperparameters and reduce overfitting to improve the model. And finally, we report the final performance of our best model, show sample predictions. We use Plotly express to have interactive and informative visualizations.

Python, Pandas, Numpy, Matplolib, Seaborn, Plotly Express, ML models.
