COVID-19 Deaths in SQL

This SQL project is focused on analyzing the data related to COVID-19 deaths and vaccinations. The data source used for this project is the "Our World in Data" website which provides information on the COVID-19 pandemic and its impact on various countries. The data used in the project includes COVID-19 deaths and vaccinations up to 01/20/2023. The project involves writing SQL queries to extract and manipulate the data stored in a relational database. The queries are used to perform various operations such as filtering, grouping, aggregating, and summarizing the data. The results of these queries will provide insights into the trends and patterns in COVID-19 deaths and vaccinations, and can be used to understand the impact of the pandemic and the effectiveness of vaccination efforts.

Diamonds: Real-World Machine Learning Model

This project is about Diamonds dataset from Kaggle, which contains information on over 54,000 diamonds, including features such as carat, cut, color, clarity, depth, table, price, x, y, and z. After conducting a thorough review of the dataset description and understanding the problem statement, we conducted exploratory data analysis to gain insights into the data. This was followed by feature engineering and the creation of a training-validation split to prepare the data for modeling. Next, we trained and evaluated various machine learning models, fine-tuning hyperparameters and reducing overfitting to enhance performance. The final step was to report the performance of the best-performing model, including sample predictions. To visually explore the data and present results, we utilized Plotly Express for interactive and informative visualizations.

Python, Pandas, Numpy, Matplolib, Seaborn, Plotly Express, ML models.

CoinMarketCap Public API

We will utilize the public API of CoinMarketCap to extract data. Using this API, we will gather information for the top 30 cryptocurrencies on CoinMarketCap over a 10-hour time frame, collecting data in 5-minute intervals. The extracted data will be saved in a CSV file for further analysis. For each cryptocurrency, we will create a linear graph showing the price changes over the 10-hour time frame (120 records for each coin). This will provide an understanding of which coins exhibit a line chart similar to that of Bitcoin. Lastly, we will calculate and display the percentage of price changes for each coin in 1 day, 7 days, 1 month, 60 days, and 90 days.

Python, requests, Pandas, Matplolib, Seaborn, Plotly Express, datetime, and time.
